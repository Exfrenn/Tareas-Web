Documentación de Endpoints

Método Ruta Descripción Body (JSON)

Pruebas Automáticas

Al cargar esta página, el script ejecuta automáticamente pruebas fetch contra cada endpoint y registra el estado y la respuesta en la consola del navegador.

Para ver los resultados, abre las herramientas de desarrollo del navegador (F12) y consulta la consola.
