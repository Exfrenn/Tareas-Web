¿Qué es Git?

Git es un sistema de control de versiones distribuido que permite rastrear cambios en el código fuente durante el desarrollo de software.

Flujo de Trabajo Básico

  1. Inicializar un repositorio
  2. Crear ramas para nuevas características
  3. Hacer commits de los cambios
  4. Subir cambios a GitHub
  5. Crear pull requests

Comandos Git Esenciales

Comando Descripción
git init Inicializa un nuevo repositorio Git
git clone Clona un repositorio existente
git add Añade cambios al área de staging
git commit Guarda los cambios en el historial
git push Sube cambios al repositorio remoto

Ramas y Colaboración

  • main/master: Rama principal del proyecto
  • feature: Ramas para nuevas características
  • bugfix: Ramas para correcciones
