Navegación de Directorios

Comando Descripción
cd Cambiar de directorio
pwd Mostrar directorio actual
ls Listar archivos y directorios

Manejo de Archivos

  • mkdir - Crear directorio
  • touch - Crear archivo
  • rm - Eliminar archivo
  • cp - Copiar archivo
  • mv - Mover/renombrar archivo

Comandos Útiles

  1. clear - Limpiar la terminal
  2. cat - Mostrar contenido de archivo
  3. grep - Buscar texto en archivos
  4. chmod - Cambiar permisos
  5. sudo - Ejecutar como superusuario

Ejemplos de Uso

                
# Crear y navegar a un directorio
mkdir proyecto
cd proyecto

# Crear archivos
touch index.html
touch style.css

# Listar archivos
ls -la
